# Contributing to Laravel WebAuthn

Thank you for considering contributing to **Laravel WebAuthn**! We welcome contributions from the community. By contributing, you agree to follow our guidelines and help us maintain a high-quality project.

## How to Contribute

### Reporting Bugs

If you find a bug, please:

1. Check the [issues](https://github.com/r0073rr0r/laravel-webauthn/issues) to see if it has already been reported.
2. If not, open a **new issue** and provide:
   - A clear description of the problem.
   - Steps to reproduce the bug.
   - Expected and actual behavior.
   - Laravel and PHP versions you are using.

### Suggesting Features

If you have an idea for a new feature:

1. Check the [issues](https://github.com/r0073rr0r/laravel-webauthn/issues) to see if it has already been requested.
2. Open a **new issue** with:
   - A clear description of the feature.
   - Use cases and examples.
   - Why it would be useful.
